Tutorial flow v6
- Removed immediate tutorial overlay behavior.
- Start screen waits for the first click.
- After click the balloon flies for 0.9 s.
- Then it brakes smoothly for 0.7 s.
- Only after braking does the swipe / obstacle tutorial appear.
- Real level obstacle waves remain frozen during the intro tutorial.
- Shield input is enabled for the learning phase.
- If the tutorial blocks are not knocked down before the timer expires, the balloon dies.
- After successful completion, normal game speed and obstacle flow resume.
